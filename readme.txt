== Description ==

Mobil seo’nun önem kazandığı bu zamanlarda, sitelerimize mobilden cihazlardan erişim artmış durumda. Bu durumda ziyaretçileriniz ile daha hızlı iletişim kurmak için bu eklentiyi geliştirdik. Sadece mobil cihazlardan erişim yapıldığında görülen telefon ve whatsapp butonları sayesinde ziyaretçilerinizi müşteriye dönüştürün. Eklenti ile ilgili destek almak için<a href="https://www.muratbutun.com/wordpress-mobil-cihazlar-icin-hizli-iletisim-eklentisi.html" target="_blank"> bu sayfayı </a>kullanabilirsiniz

== Installation ==

Eklentiyi indirip /wp-content/plugins dizinine atın. WordPress eklentiler ekranından aktif edin. İletişim bilgilerinizi girmek için Eklentiler > Mobil İletişim sayfasına tıklayın. Eklenti kurulumu ile ilgili detaylı bilgi için <a href="https://www.muratbutun.com/egitim/wordpress-eklenti-yukleme" target="_blank">tıklayınız.</a>
